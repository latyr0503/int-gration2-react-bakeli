Comment changer la couleur de la balise HR avec CSS

juillet 11, 2019fvrier 10, 2020 Amine KOUIS Aucun commentaire

Vous pouvez simplement utiliser la proprit background-color en combinaison avec les proprits height et border pour changer la couleur par dfaut de la balise <hr>.

Code source :
<!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8">
		<title>Changer la couleur de la balise HR avec CSS</title>
		<style>       
			hr{
				height: 3px;
				background-color: #ff7d95;
				border: none;
			}
		</style>
	</head>
	<body>
		<h1>Titre</h1>
		<hr>
		<p>Lorem ipsum dolor sit amet, consectetur adipiscing elit.</p>
	</body>
</html>
